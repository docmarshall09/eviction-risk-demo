Model Summary

What this demo predicts and what it does not.

This page explains the label, features, data source, and practical limitations.

Label

The model predicts whether a county will land in the next year’s top quartile of filing_rate.

Features

  • lag_1
  • lag_3_mean_obs
  • lag_5_mean_obs
  • years_since_last_obs

Data Source

Princeton Eviction Lab county-level observed/validated data, sourced from county_proprietary_valid_2000_2018.csv.

Limitations

  • Outcomes end at the 2018 outcome-year.
  • Predictions are county-level only.
  • This is a demo and not a causal model.
  • Scores reflect historical correlations, not guarantees.
  • Do not use this demo for high-stakes decisions.
