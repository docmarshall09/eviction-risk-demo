API Quickstart

Endpoints and copy/paste examples.

This frontend uses same-origin API calls to the FastAPI backend. Full interactive docs are available at /docs.

Endpoints

  • GET /health
  • GET /metadata
  • POST /score
  • POST /score/batch

/score (county_fips only)

curl -X POST http://127.0.0.1:8000/score \
  -H "Content-Type: application/json" \
  -d '{"county_fips":"39049"}'

/score (county_fips + as_of_year)

curl -X POST http://127.0.0.1:8000/score \
  -H "Content-Type: application/json" \
  -d '{"county_fips":"39049","as_of_year":2015}'

/score/batch (2 counties, one with year omitted)

curl -X POST http://127.0.0.1:8000/score/batch \
  -H "Content-Type: application/json" \
  -d '[{"county_fips":"39049","as_of_year":2015},{"county_fips":"06037"}]'
