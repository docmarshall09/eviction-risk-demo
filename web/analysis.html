Backtest Analysis

How the model performed on held-out outcome years.

These charts summarize relative performance and calibration from a leakage-safe holdout setup.

Key findings

  • The model performed strongly in holdout backtests and separated higher-risk counties well.
  • A simple naive baseline using only lag_1 remained close in performance.
  • That gap pattern suggests eviction filing risk has persistence year-over-year.
  • Calibration trends are directionally reasonable but not perfect at every score band.
  • Outcomes in this dataset end in 2018, so results reflect historical behavior only.
