Engineering View

Technical scope

This demo includes an end-to-end ML workflow: yearly model training, artifact packaging, FastAPI scoring endpoints, a static web UI, and containerized deployment on Render.

Stack summary

  • FastAPI serves the API and the static demo UI.
  • Yearly logistic regression (scikit-learn) is trained at Docker build time.
  • Model artifacts are bundled into the runtime image for fast scoring.
  • Hosted on Render; deploy triggers on GitHub push.
