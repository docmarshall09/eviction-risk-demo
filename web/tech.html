API usage

Base URL: https://eviction-risk-demo.onrender.com · Docs: https://eviction-risk-demo.onrender.com/docs

GET /health

curl -X GET https://eviction-risk-demo.onrender.com/health

POST /score (county_fips only)

curl -X POST https://eviction-risk-demo.onrender.com/score \
  -H "Content-Type: application/json" \
  -d '{"county_fips":"39049"}'

POST /score (county_fips + as_of_year)

curl -X POST https://eviction-risk-demo.onrender.com/score \
  -H "Content-Type: application/json" \
  -d '{"county_fips":"39049","as_of_year":2015}'

POST /score/batch (2 items)

curl -X POST https://eviction-risk-demo.onrender.com/score/batch \
  -H "Content-Type: application/json" \
  -d '[{"county_fips":"39049","as_of_year":2015},{"county_fips":"06037"}]'
